# Multimodal Deep Learning Models for Anaesthesia Monitoring

This repository contains the code for training and evaluating our deep learning models.

## Project Structure
* `data/`: Contains raw `.vital` and processed `.csv` data (ignored by git).
* `models/`: Contains the Transformer and future model architectures and saved weights.
* `src/`: Python and Bash scripts for data processing, training and testing.
* `notebooks/`: Jupyter notebooks for prototyping
